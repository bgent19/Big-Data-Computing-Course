# Big Data Computing Course
This is a collection of my material for giving a course on Big Data Computing in an undergraduate environment. It consists of course content and associated lab excercises for students or personal learning. My goal was to create a course that was free to use based on the restrictions I had when adminsitering this course.

## Target Audience
My target audience was undergarduate Data Science Students. It was a required 400-level course where they would have taken a pre-requisiste course on tradtional database storage and machine-learning.

## Content Layout
There are 3 Primary units in this class:
 1. Big Data Storage
 2. Big Data Computation
 3. Straming Systems

There are associated labs for each unit for students to see in practice what was taught in lecture.

I had 50-minute lecture periods when I gave this course, so you will see in the class HTML files specific "Lecture Break" comments.  That pacing is for 50 minute class periods.

These three units took me about 11 weeks to complete.  I then added a substantial project at the end of the semester that would mimic the required senior capstone project the students would perfrom in the following semester. 

## Surrounding Infrastructure
The HTML material was designed to be put into Jeff Kenney's [Calendar system](https://github.com/jskenney/calendar.git) for deployment of the course. All labs were designed to be run on any machine that can run Docker.
